# GeekTime 算法训练营22期

- 切题四件套
  - 明确题意
  - 思考所有解法, 比较时空复杂度,找到最优解
  - 写代码
  - 测试

- 五毒神掌
  - 第一遍
    - 5分钟读题 + 思考
    - 超时直接看解法(多解),比较优劣
    - 背诵默写好的解法
  - 第二遍
    - 自己写 --> leecode 提交
    - 每种解法都要写一遍, 体会优劣
  - 第三遍
    - 过了一天后, 重复昨天的题目
    - 对不熟悉的解法进行专项训练
  - 第四遍
    - 过了一周后,重复"第三篇"的行为
  - 第五遍
    - 面试前,再做一遍
